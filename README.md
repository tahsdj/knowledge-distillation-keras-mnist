# knowledge-distillation-keras-mnist
It is a simple demo for using knowledge distillation in mnist dataset

# Results

* Temperature = 7
* Alpha = 0.9

| Teacher | Student | KD |
| -------- | -------- | -------- |
| 99.5(CNN)     | 98.41 (1 layer FC net)   | 98.89(+0.48)    |
